# Jira Agile Workflow – Opencart E-commerce Project

## 📌 Project Overview
This project demonstrates how Agile methodology is implemented using Jira for managing an e-commerce application.

## 🛠 Tools Used
- Jira
- Scrum Framework

## 🚀 Workflow Implemented

- Created a project using Scrum template
- Created backlog and added user stories
- Planned Sprint 0
- Moved tasks from backlog to sprint
- Created subtasks for development and testing
- Started sprint execution
- Tracked tasks using Scrum board (To Do → In Progress → Done)
- Completed the sprint

## 🔄 Agile Process
This project follows an iterative approach where tasks move from backlog to sprint and are completed in cycles.
